# Благодарности

Проект опирается на открытые библиотеки Python и публичные источники рыночных данных.

## Основные технологии

- `pandas` и `numpy` для обработки временных рядов.
- `matplotlib` для рендера графиков.
- `imageio-ffmpeg` для сборки MP4.
- `apimoex` и MOEX ISS для российских инструментов.
- `yfinance` для глобальных инструментов.
- `requests` для Telegram Bot API и HTTP-запросов.
- `pyarrow` для parquet-кэша.
- `pytest`, `ruff` и `compileall` для проверок качества.
